When you find useful links to Web resources that help you in your preparation of the Learning Outcomes Contract, please share them with your Learning Set (and future learners) using Delicious.

To use this you need a Delicious account, which you can set up yourself if you don't already have one. If you are a Firefox user, we recommend that you use the free plug-in, which gives you an easy way to add shared bookmarks straight from your browser. 

Also there a is a Drop.io service provided where you can drop your files to be shared & also review them.
