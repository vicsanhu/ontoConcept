Sharing useful resources

For storing and sharing the Learning Outcomes Negotiation you can use Deliciousto share resources from the webwhich you find useful. These will be shared not only with your learning set, but also with other learning sets carrying out this activity now and in the future. To use this you need a Delicious account, which you can set up yourself if you don't already have one. If you are a Firefox user, we recommend that you use the free plug-in, which gives you an easy way to add shared bookmarks straight from your browser.

If you want to share files then use the drop-boxwidget. This works with the drop.io service. An account has already been setup to use this. To view files that have already been uploaded you should use the login name "guest".
