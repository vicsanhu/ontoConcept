ReCourse Developers Notes 
----------------------------

This is an Eclipse 3.5 RCP application.  You will need a minimum:

1.  Eclipse 3.5
2.  Eclipe GEF 3.5
3.  Eclipse RCP Delta pack.


Dependant modules:

org.tencompetence.jdom						JDOM XML library
wp6/org.eclipse.epf.richtext				Rich Text Editor
wp6/org.eclipse.epf.imsldmodel				LD Model
wp6/org.tencompetence.ldauthor 				The Main LD Editor
wp6/org.tencompetence.ldauthor.fedora 		Fedora DB handler
wp6/org.tencompetence.ldauthor.opendock		OpenDocument Search module
wp6/org.tencompetence.ldpublisher			Publish to CopperCore Server module
wp6/org.tencompetence.ldvisualiser			Visualiser
wp6/org.tencompetence.qtieditor				QTI Editor
wp6/org.tencompetence.widgetadvert			CopperCore Widget UI library


CVS contains a branch, "Version_1_0_1.  This is the first GEF-based version of the Editor.
